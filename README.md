## 구현 기능 목록

1. 경주할 자동차 이름 및 시도 회수 입력
    - 주어지는 이름이 5자 이하인지 확인
    - 주어지는 이름 문자열이 0이 아닌지 확인
    - 주어지는 이름 문자열이 문자열(1~6자리)과 쉼표로 구성되어있는지 확인 
    - 게임진행회수가 0보다 큰 정수인지 확인
2. 경주 자동차 이름 및 초기 위치 설정
    - 주어지는 이름 문자열에 중복된 이름이 있는지 확인
3. 각 경주마다 전진하는 조건 random 생성
    - 랜덤값이 1~9 사이의 값인지 체크
4. 자동차 전진/멈춤 판단 로직
5. 경주 완료시 우승자 구하기
